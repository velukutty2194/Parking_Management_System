the program is dynamically coded
hence can flex acording to the users requirement

once the code has run
it will display the user 4 options to choose by entering any number from 1 to 4
1)the first option is to display the parking lot
	after this option it will display the parking lot
2)the second option is to park the vehicle
	-this option it will ask the user to enter the lot number the vehicle has to be alloted
3)the third option is to remove the vehilce 
	-this option will further ask the user to enter the slot number from which the vehicle has to be removed from
	if the user enters the slot number which is already empty,it will display that the slot is not occupied
4)the fourth option is to exit from the code
